I fix the WooCommerce problems that keep merchants stuck. Happiness Engineer and Woo Marketplace Product Ambassador at [Automattic](https://automattic.com). I work with PHP, JavaScript, TypeScript, REST APIs, and whatever it takes to find the root cause.

---

### What I Do

**Debug the hard stuff.** Subscriptions that fail silently. Payment gateways that stop processing. Checkout pages that break after updates. I dig into logs, trace the code, and figure out why.

**Build tools from real problems.** The issues I see in support often become the products I develop:

| Project | What it does |
| --- | --- |
| [HappyAccess](https://github.com/shameemreza/happyaccess) | Secure temporary admin access via OTP and magic links |
| [SnipDrop](https://github.com/shameemreza/snipdrop) | Tested WooCommerce snippets with one-click install |
| [Resubscribe Controls](https://github.com/shameemreza/resubscribe-controls-for-wcsubs) | Manage resubscription behavior for WooCommerce Subscriptions |
| [Kairo Theme](https://github.com/shameemreza/kairo) | A clean WordPress theme |

**Review plugins for WooCommerce.com.** Code quality, security, and UX before they go live.

**Write about what I learn.** Real support cases turned into practical solutions at [shameem.blog](https://shameem.blog).

**Contribute to WordPress.** Plugin forums for WooCommerce (7M+ installs), WooPayments, Stripe, PayPal Payments. Also active as a plugin developer, theme author, pattern author, translation contributor, and more on [WordPress.org](https://profiles.wordpress.org/shameemreza/).
